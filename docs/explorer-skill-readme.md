# Explorer Skill 文档包

## 文件说明
- **explorer-skill.zip**: 包含完整的Explorer技能文档和相关文件

## 包含内容
```
explore/
├── SKILL.md                    # 主要技能文档
├── examples/                   # 使用示例
│   ├── basic-extraction.sh     # 基础数据提取脚本
│   ├── advanced-extraction.sh  # 高级数据提取脚本
│   ├── extraction-config.json  # 提取配置示例
│   └── advanced_extraction.py  # Python高级提取脚本
└── references/                # 参考文档
    ├── api-reference.md       # API参考文档
    ├── advanced.md           # 高级使用指南
    └── pagination.md         # 分页处理指南
```

## 用途
这个zip包包含了NavCLI Explorer技能的所有文档和示例，可用于：
- 学习和理解Explorer技能的使用方法
- 快速上手数据提取和网站探索
- 参考示例进行自定义开发

## 安装和使用
1. 解压zip文件
2. 阅读 `explore/SKILL.md` 了解基础用法
3. 参考 `examples/` 目录下的示例脚本
4. 查看 `references/` 目录下的详细文档
